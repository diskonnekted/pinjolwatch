## 💡 TIPS PENTING

1. **Selalu cek legalitas pinjol** di website OJK: https://ojk.go.id
2. **Jangan berikan akses** kontak, galeri, atau lokasi ke aplikasi pinjol
3. **Dokumentasikan semua ancaman** untuk bukti hukum
4. **Lapor segera** jika mengalami teror atau penyebaran data
5. **Jangan bayar bunga di atas batas legal** (maksimal 0,1%/hari untuk konsumtif)
